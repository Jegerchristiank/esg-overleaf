\subsection{Rapportens opbygning}
\label{subsec:rapportens-opbygning}

Rapporten er opbygget som en samlet argumentk\ae de fra kontekst til konklusion:
\begin{enumerate}
  \item Afsnit \ref{sec:kontekst-og-rammer} etablerer konteksten for \ESG-rapportering i sundhedssektoren og de regulatoriske rammer.
  \item Afsnit \ref{sec:teoretisk-ramme} pr\ae senterer den teoretiske ramme for standardisering, governance og \o konomisk relevans.
  \item Afsnit \ref{sec:metode} beskriver forskningsdesign, datagrundlag og caseafgr\ae nsning.
  \item Afsnit \ref{sec:software-og-mvp} forklarer \MVP'ens form\aa l, arkitektur og datagrundlag som operationalisering af kravene.
  \item Afsnit \ref{sec:analyse} anvender teori p\aa empiri og case for at vurdere v\ae rdiforslag og implikationer.
  \item Afsnit \ref{sec:diskussion} diskuterer sammenh\ae nge, begr\ae nsninger og implikationer for \SMV'er.
  \item Afsnit \ref{sec:konklusion} samler svarene p\aa forskningssp\o rgsm\aa lene, og \ref{sec:perspektivering-og-anbefalinger} peger p\aa videre arbejde og anbefalinger.
\end{enumerate}
