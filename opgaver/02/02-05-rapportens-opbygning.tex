\subsection{Rapportens opbygning}
\label{subsec:rapportens-opbygning}

Rapporten er opbygget som en samlet argumentkæde fra indledning til konklusion:
\begin{enumerate}
  \item Afsnit \ref{sec:indledning} introducerer problemfelt, formål og struktur.
  \item Afsnit \ref{sec:kontekst-og-rammer} etablerer konteksten for ESG-rapportering i sundhedssektoren og de regulatoriske rammer.
  \item Afsnit \ref{sec:teoretisk-ramme} præsenterer den teoretiske ramme for standardisering, governance og økonomisk relevans.
  \item Afsnit \ref{sec:metode} beskriver forskningsdesign, datagrundlag og caseafgrænsning.
  \item Afsnit \ref{sec:software-og-mvp} forklarer MVP'ens formål, arkitektur og datagrundlag som operationalisering af kravene.
  \item Afsnit \ref{sec:analyse} anvender teori på empiri og case for at vurdere værdiforslag og implikationer.
  \item Afsnit \ref{sec:diskussion} diskuterer sammenhænge, begrænsninger og implikationer for SMV'er.
  \item Afsnit \ref{sec:konklusion} samler svarene på forskningsspørgsmålene, og \ref{sec:perspektivering-og-anbefalinger} peger på videre arbejde og anbefalinger.
  \item Bilagene er samlet i \ref{app:bilag} og understøtter sporbarhed og dokumentation.
\end{enumerate}
